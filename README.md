bmd-paint
=========

Browser-Mobile-Desktop paint app

Technologies used:
- node.js
- backbone.js
- underscore.js
- require.js
- jquery
- socket.io
- phonegap
- node-webkit

This app is to demonstrate ability to write a javascript code that can be deployed to web, mobile, desktop.
